## 评分分布预估

基于6因子模型（技术20%+情绪10%+板块35%+资金20%+风险15%）：

### 理论分布
- **平均评分**: 约50-55分（中心基准）
- **标准差**: 约10-15分
- **正态分布**: 大部分股票集中在40-70分区间

### 阈值建议

| 阈值 | 选中比例 | 适用场景 |
|-----|---------|---------|
| **60分** | 前30% | 宽松，适合震荡市 |
| **65分** | 前20% | 标准，当前使用 |
| **70分** | 前15% | 严格，适合牛市 |
| **75分** | 前10% | 极严，适合强势市场 |

### 动态调整建议

```
大盘强势（涨幅>1%）: 阈值 70分
大盘震荡（-1%~1%）: 阈值 65分  <- 当前使用
大盘弱势（<-1%）: 阈值 60分（或减少选股数量）
```
