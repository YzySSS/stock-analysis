## 二、选股逻辑

### 2.1 基础过滤
- 股价范围：5元 - 150元
- 成交额：≥5000万（turnover >= 0.5亿）
- 排除无PE或ROE数据的股票

### 2.2 因子体系（2因子简化版）

| 因子 | 定义 | 权重 | 方向 |
|------|------|------|------|
| **Quality（质量）** | ROE得分（roe_score） | 50% | 正向 |
| **Valuation（估值）** | PE得分（pe_score） | 50% | 正向 |

### 2.3 评分方法
```python
# Z-score标准化
quality_z = (ROE - ROE均值) / ROE标准差
valuation_z = (PE - PE均值) / PE标准差

# 加权得分
score = 50 + (quality_z * 0.5 + valuation_z * 0.5) * 15
```

### 2.4 选股规则
- 阈值：score >= 50
- 每日最多选5只
- 冷却期：5天（同一股票5天内不重复选）

---

## 三、交易规则

### 3.1 买入时机
- 选股日**开盘价**买入

### 3.2 卖出时机（固定持有期）
| 持有期 | 卖出价格 | 说明 |
|--------|----------|------|
| 1日 | 次日开盘价 | T+1开盘卖出 |
| 3日 | 第3日收盘价 | 持有3天后收盘卖出 |
| 5日 | 第5日收盘价 | 持有5天后收盘卖出 |

### 3.3 成本设置
- 佣金：0.05%（双边）
- 印花税：0.1%（卖出）
- 滑点：0.1%（双边）
- **总成本：0.25%/笔**

## 四、回测方式

### 4.1 回测逻辑
```python
for each trading_day:
    # 1. 选股
    candidates = 符合条件的所有股票
    picks = score排名前5且score>=50的股票
    
    # 2. 模拟买入
    for pick in picks:
        entry_price = 当日开盘价
        
    # 3. 计算收益（固定持有期后卖出）
    exit_price = hold_days后的价格
    gross_return = (exit_price - entry_price) / entry_price
    net_return = gross_return - 0.0025  # 扣除成本
```

### 4.2 当前累计收益计算方式
```python
# 方式1：复利连乘（当前使用）
cum = 1.0
for each trade:
    cum *= (1 + net_return)
total_return = cum - 1

# 例如：250笔交易，平均收益0.01%
# cum = 1.0001^250 ≈ 1.025
# total_return = 2.5%
```

## 六、我们的疑问

### 问题1：累计收益计算方式是否正确？
当前使用**复利连乘**方式：
```python
cum = 1.0
for r in net_returns:
    cum *= (1 + r)
total_return = cum - 1
```

**疑问**：
1. 这种计算方式假设每次交易都满仓，且收益复利滚动，是否合理？
2. 对于选股策略评估，是否应该用**算术平均**而非**几何平均**？
3. 是否应该考虑**资金利用率**（并非每日都满仓）？

2. 应该如何处理**资金占用**问题（比如选5只股，但资金没有全部用完）？
