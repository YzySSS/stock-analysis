### 4. 完整成本模型实现

```python
class RealCostModel:
    """真实成本计算模型"""
    
    def __init__(self):
        self.commission_rate = 0.00025  # 万2.5
        self.min_commission = 5         # 最低5元
        self.stamp_duty_rate = 0.0005   # 千0.5
        self.transfer_rate = 0.00001    # 过户费
        self.base_slippage = 0.002      # 基础滑点0.2%
    
    def calculate_roundtrip_cost(self, amount, avg_volume):
        """计算一次完整交易（买+卖）的真实成本"""
        
        # 1. 固定成本
        buy_commission = max(amount * self.commission_rate, self.min_commission)
        sell_commission = max(amount * self.commission_rate, self.min_commission)
        stamp_duty = amount * self.stamp_duty_rate  # 仅卖出
        transfer_fee = amount * self.transfer_rate * 2  # 买卖各一次
        
        fixed_cost = buy_commission + sell_commission + stamp_duty + transfer_fee
        
        # 2. 滑点成本（买卖双边）
        position_ratio = amount / avg_volume
        if position_ratio < 0.01:
            slip_rate = 0.001
        elif position_ratio < 0.05:
            slip_rate = 0.002
        else:
            slip_rate = 0.005
        
        slippage_cost = amount * slip_rate * 2  # 买卖各一次
        
        # 3. 总成本
        total_cost = fixed_cost + slippage_cost
        cost_ratio = total_cost / amount
        
        return {
            'fixed_cost': fixed_cost,
            'slippage_cost': slippage_cost,
            'total_cost': total_cost,
            'cost_ratio': cost_ratio,  # 成本占本金比例
            'real_buy_price': buy_price * (1 + slip_rate + self.commission_rate),
            'real_sell_price': sell_price * (1 - slip_rate - self.commission_rate - self.stamp_duty_rate)
        }
```

## 真实成本影响预估

### 成本计算（基于V12当前参数）

**假设条件**：
- 本金：10万元
- 交易频率：每日满仓换股（250次/年）
- 平均成交额：假设5000万（中等流动性，0.2%滑点）

**年化成本计算**：
```
固定成本/轮 = 25+25+50+2 = 102元 = 0.102%
滑点成本/轮 = 100000 × 0.002 × 2 = 400元 = 0.4%
总成本/轮 = 502元 = 0.502%

年化成本 = 0.502% × 250 = 125.5%
```

**修正后收益预估**：
```
理论收益：+159%（2年）
扣除成本：159% - 125.5% × 2 = -92%
```

**结论**：如果不改变交易频率，真实收益可能为**大幅亏损**。
