# V12策略 V6 完整分析报告

## 一、策略核心逻辑

### 1.1 因子定义（7因子）

| 因子 | 计算方法 | 权重 |
|------|----------|------|
| **trend** | MA20斜率年化 = (MA20_now - MA20_prev) / MA20_prev × 252 | 18% |
| **momentum** | 20日涨幅 = (P_today - P_20days_ago) / P_20days_ago × 100 | 13% |
| **quality** | ROE（行业中性化后）或 20 - 波动率×0.5 | 18% |
| **sentiment** | 前一日涨跌幅 pct_change | 13% |
| **valuation** | PE得分（负值，越高估值越低） | 18% |
| **liquidity** | ln(turnover + 1) | 10% |
| **size** | -ln(market_cap)，小市值偏好 | 10% |

### 1.2 评分公式

```python
# Z-score标准化
zscore = (factor_value - mean) / std
zscore = clip(zscore, -3, 3)

# 加权得分
weighted_z = Σ(zscore_i × weight_i)
final_score = 50 + weighted_z × 15
final_score = clip(final_score, 0, 100)
```

### 1.3 选股规则

1. **基础过滤**：股价5-150元，换手率≥0.5%，非ST，MA20向上且价格≥MA20×0.9
2. **达标即选**：score ≥ 55分入选，最多5只
3. **行业约束**：单行业占比≤30%（即最多1-2只）
4. **冷却期**：买入后3天内不重复买入同一股票

### 1.4 交易模型

- **T+1制度**：选股日收盘后运行，次日开盘价买入，次日收盘价卖出
- **仓位管理**：市场强度(0-100) → 仓位20%-100%
- **动态权重**：
  - 强势(>60): 进攻性因子×1.05，风险因子×0.90
  - 弱势(<40): 进攻性因子×0.95，风险因子×1.20
- **止损**：-5%触发止损
- **成本**：佣金0.05% + 印花税0.1% + 滑点0.1%

### 1.5 IC/IR优化

每20个交易日根据历史IC调整因子权重：
```python
IC = corrcoef(zscore_factor, future_return)
weight_adjustment = 1 + IC × 0.3
```
