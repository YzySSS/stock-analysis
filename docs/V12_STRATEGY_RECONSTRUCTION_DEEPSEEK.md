# DeepSeek 策略重构分析报告

## 总体评价
**该策略是一个过度暴露于小盘成长风格、且缺乏有效风控的“伪多因子”策略，其结构复杂但内核脆弱，在熊市中完全失效，2026年高收益极大概率是过拟合或幸存者偏差的体现，不具备实盘价值。**

## 1. 因子有效性诊断
### 1.1 有效因子（建议保留/改造）
- **quality (ROE质量分)**：**保留，但需改进。** 盈利质量是长期有效的阿尔法来源，但单期ROE稳定性差。建议改为“ROE-TTM的行业分位数”或结合杜邦分析（ROE稳定性、盈利质量）。
- **valuation (估值)**：**保留，且应加强。** 估值是控制回撤的关键因子，尤其在弱势市场。当前定义（-PE*0.5 - PB*10）粗糙。建议拆分为独立的PE-TTM、PB分位数因子，并在弱势市场大幅提高其权重。

### 1.2 无效/问题因子（建议删除/修改）
- **trend 与 momentum**：**合并并改造。** 两者高度共线性（都是趋势类），且在熊市（如24-25年）同时失效，是造成巨大回撤的主因。建议合并为一个**中长期动量/趋势因子**，例如“过去60日收益率 - 过去5日收益率”（捕捉中期动量、过滤短期噪音），并降低其权重。
- **sentiment (前一日涨跌幅)**：**坚决删除。** 这是典型的噪音因子，无预测能力，13%的权重严重拖累策略信噪比，是导致胜率低下的重要原因。
- **size (小市值因子)**：**调整暴露方式，不可简单做多。** 小市值因子在A股长期有效，但波动巨大（见84.6%回撤）。10%的定向做多权重导致策略在熊市中小盘暴跌时完全暴露。**建议改为行业市值中性化处理，或在组合构建时约束市值暴露，而非作为一个正向打分因子。**

### 1.3 缺失的关键因子（建议添加）
- **反转因子 (Reversal)**：A股市场存在显著的短期反转效应（如过去1个月表现差的股票，未来短期内可能反弹）。这是**提升胜率最关键的因子之一**。可添加“过去20日收益率（负向）”。
- **波动率因子 (Volatility)**：低波动率因子在全球市场长期有效，能显著降低组合回撤。例如“过去60日收益率波动率的倒数”。
- **改进的流动性因子**：当前“成交额对数”过于简单。建议改为“阿米希德流动性比率”或“成交额/流通市值的行业分位数”，更能捕捉真实的交易活跃度与冲击成本。

## 2. 胜率提升方案
### 2.1 胜率分析
当前胜率45.5%<54%盈亏平衡点，核心问题：
1.  **噪音因子(sentiment)占比高**，引入大量随机交易。
2.  **趋势/动量因子在震荡市和熊市产生大量假信号**，导致止损卖出。
3.  **缺乏能提供高胜率入场信号的因子**，如短期反转。
4.  **持仓周期过短(2天)**，未给予阿尔法信号足够的时间发酵，更像是一个“T+2”的博弈策略，摩擦成本极高。

### 2.2 具体改进措施
- **删除sentiment因子**：立即可提升策略整体信噪比。
- **加入短期反转因子(如过去20日收益率负向)**：旨在捕捉超跌反弹的机会，这类信号胜率通常较高。
- **延长持仓周期**：从2天延长至**5-20个交易日**，让基本面（quality）和估值（valuation）因子的效力得以体现，减少频繁交易带来的摩擦损失和随机性。
- **提高选股得分阈值**：从55分提高至**60-65分**，只交易信号最强的股票，牺牲部分交易次数以换取更高胜率。
- **引入“因子拥挤度”排查**：避免买入短期内因子得分已急剧飙升（可能过度拥挤）的股票。

## 3. 回撤控制方案
### 3.1 回撤来源分析
84.6%的回撤是灾难性的，主要来自：
1.  **风格暴露集中**：对“小市值”和“趋势/动量”因子的单向做多暴露，在2024-2025年小盘股、成长股双杀行情中被“连锅端”。
2.  **无止损机制**：买入后固定持有到期，无论期间跌幅多大。
3.  **市场环境判断失效**：`market_strength`指标未能有效识别熊市并降低风险暴露（2024年仍有100笔交易）。
4.  **仓位管理粗糙**：90%的固定高仓位在熊市中放大损失。

### 3.2 风控机制设计
- **硬性个股止损**：入场后设置**-8%** 的硬止损线。这是控制回撤的第一道且最有效的防线。
- **改进的市场状态过滤**：不要仅基于`market_strength`调整因子权重，而应直接**控制开仓**。例如，当沪深300指数低于其200日均线（熊市标志）时，**停止开仓或仅允许开仓1-2只股票**。
- **严格的组合风险控制**：
    - **行业暴露上限**：从30%降至**15%-20%**。
    - **市值暴露控制**：确保组合相对基准（如中证500）的市值暴露不超过±0.3。
- **动态仓位管理**：将90%固定仓位改为基于市场波动率的动态仓位。例如，仓位 = 基准仓位(如60%) * (基准波动率/当前市场波动率)。

## 4. 策略重构方案
### 4.1 推荐的新因子组合 (4因子核心)
遵循“逻辑驱动、简单有效、风险可控”原则，将因子精简为4个核心因子。

| 因子 | 权重 | 定义（建议） | 理由与说明 |
| :--- | :--- | :--- | :--- |
| **质量 (Quality)** | 30% | ROE-TTM的行业稳定性评分（过去8季度的标准差倒数） | 核心阿尔法来源，提供长期稳定性和安全边际。高权重。 |
| **估值 (Value)** | 30% | (PE-TTM分位数 + PB分位数) / 2 的负向得分 | 控制回撤的“压舱石”，在弱势市场权重可自动提升（通过分位数加权）。 |
| **反转 (Reversal)** | 25% | -1 * 过去20日收益率（行业中性化后） | **提升胜率的关键**，捕捉短期错误定价的修正。 |
| **波动率 (LowVol)** | 15% | -1 * 过去60日收益率波动率（行业中性化后） | **降低回撤的关键**，偏好稳定股票，天然规避高风险品种。 |

**注1**：所有因子均需进行**行业中性化**处理（计算行业内的Z-score），然后再合成总分，以剥离行业beta。
**注2**：**不再保留独立的trend/momentum/size/liquidity因子**。流动性通过在选股池中设置最低成交额门槛（如1亿元）来控制。市值风险通过组合暴露约束来控制。

### 4.2 新的选股与交易规则
1.  **股票池过滤**：全A股，去ST/退市/北交所，价格>5元，**过去20日均成交额>1亿元**。
2.  **因子计算与合成**：按上表计算4因子，行业中性化后，按固定权重合成综合得分。
3.  **选股**：
    - 选取综合得分最高的前**30**只股票。
    - 进行组合优化：以**最大化综合得分**为目标，约束条件包括：行业权重≤15%，个股权重≤5%，**市值暴露相对中证500±0.3以内**。
    - 输出最终**5-8**只股票作为持仓。
4.  **交易**：
    - **买入**：T+1日开盘。
    - **卖出**：
        - **条件1（止损）**：持仓亏损触及**-8%**，立即卖出。
        - **条件2（止盈/换仓）**：固定持仓**10个交易日**后，于下一调仓日卖出。
        - **条件3（信号失效）**：若个股在持仓期内，其综合得分跌出全市场前50%，提前卖出。
5.  **仓位与风控**：
    - **基准仓位60%**。
    - **市场过滤**：若沪深300收盘价低于其120日均线，仓位上限降至**30%**；若低于200日均线，**暂停开仓**，仅持有现有仓位并执行止损。

### 4.3 预期效果
- **胜率**：45.5% → **目标52%-58%**。通过引入反转因子、提高阈值、延长持仓周期实现。
- **回撤**：84.6% → **目标25%-35%**。通过止损、低波因子、市值/行业约束、市场过滤四重风控实现。
- **收益**：放弃不切实际的2026年超高收益预期。**目标年化收益率15%-25%**，夏普比率>1.2。策略目标从“高收益”转变为“高胜率、中收益、低回撤”的稳健型产品。
