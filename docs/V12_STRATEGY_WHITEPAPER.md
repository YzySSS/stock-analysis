## 2. 因子体系

### 2.1 六大因子详解

| 因子 | 权重 | 计算方式 | 经济含义 |
|------|------|----------|----------|
| **Trend** (趋势) | 20% | MA20年化斜率 = (MA20_now - MA20_prev) / MA20_prev × 252 | 中期趋势方向 |
| **Momentum** (动量) | 15% | 20日涨幅 = (Close_t - Close_t-20) / Close_t-20 × 100 | 价格动量 |
| **Quality** (质量) | 20% | ROE_clean（行业中性化后）或 20 - 波动率×0.5 | 公司质量 |
| **Sentiment** (情绪) | 15% | 前一日涨跌幅 | 短期情绪 |
| **Valuation** (估值) | 20% | PE_score（行业中性化后，负值表示低估值） | 估值水平 |
| **Liquidity** (流动性) | 10% | ln(成交额 + 1) | 交易活跃度 |

### 2.2 Z-score标准化

```python
# 每个因子在股票池内进行标准化
zscore = (原始值 - 均值) / 标准差
zscore = clip(zscore, -3, 3)  # 限制极端值

# 最终得分
加权Z = Σ(zscore_i × weight_i)
总分 = 50 + 加权Z × 15  # 映射到0-100分
```

### 2.3 动态权重调整 (P1优化)

根据市场强度动态调整因子权重：

**强势市场 (>60)**:
- Trend: 22% (+2%)
- Momentum: 17% (+2%) 
- Quality: 21% (+1%)
- Sentiment: 16% (+1%)
- Valuation: 18% (-2%)
- Liquidity: 6% (-4%)

**弱势市场 (<40)**:
- Trend: 18% (-2%)
- Momentum: 13% (-2%)
- Quality: 19% (-1%)
- Sentiment: 14% (-1%)
- Valuation: 22% (+2%)
- Liquidity: 14% (+4%)

## 3. 市场强度指数

### 3.1 计算公式

市场强度 = 趋势40% + 宽度30% + 成交量20% + 情绪10%

| 成分 | 数据来源 | 计算方式 |
|------|----------|----------|
| 趋势40% | 沪深300 | MA20斜率映射到0-100 |
| 宽度30% | 全市场 | 上涨股票比例 |
| 成交量20% | 沪深300 | 当前成交量/20日均量 |
| 情绪10% | 沪深300 | 前一日涨跌幅×5 + 50 |

### 3.2 仓位管理

```python
仓位比例 = 20% + (市场强度 / 100) × 80%
# 强度0 → 20%仓位
# 强度50 → 60%仓位
# 强度100 → 100%仓位
```

## 4. 交易规则

### 4.1 选股流程

1. **硬性过滤**:
   - 股价: 5-150元
   - 成交额: ≥5000万
   - 排除ST/*ST/退市股
   - 价格在MA20×0.9之上

2. **因子评分**:
   - Z-score标准化
   - 动态权重加权
   - 阈值: ≥55分入选

3. **数量限制**:
   - 最多5只 × 仓位比例
   - 按分数排序取Top N

### 4.2 交易模型 (T+1)

```
Day T 收盘后: 选股 → 生成信号
Day T+1 开盘: 买入 (开盘价 + 滑点)
Day T+1 收盘: 卖出 (收盘价 - 滑点)
```

### 4.3 成本模型

| 成本项 | 费率 | 说明 |
|--------|------|------|
| 佣金 | 0.05% | 双边 |
| 印花税 | 0.10% | 卖出单边 |
| 滑点 | 0.10% | 基础滑点 |
| 冲击成本 | 变动 | 大额交易附加 = 占成交额比例 × 0.1 |
| **总成本** | ~0.20% | 每笔交易 |

### 4.4 止损机制

- 止损线: -5%
- 触发条件: 当日最低价 ≤ 买入价 × 0.95
- 止损卖出价: 止损线价格

## 6. 数据预处理

### 6.1 基本面数据清洗

**原始数据问题**:
- PE极值: 3.86 - 980 (均值79.75)
- ROE极值: -23.52 - 43.19

**清洗流程**:
1. **MAD去极值**: 中位数绝对偏差法，限制在±3倍MAD
2. **行业中性化**: Z-score标准化 (值 - 行业均值) / 行业标准差
3. **PE转得分**: 负值表示低估值好 (高分 = 低估值)

**清洗效果**:
- PE: 均值79.75→55.32, 最大值980→133
- ROE: 极值43/-23→15/-12

### 6.2 新股处理

- 排除上市首日数据（涨跌幅可能>500%）
- 需要至少21日历史数据才纳入选股池
