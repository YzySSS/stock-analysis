## 现有因子体系

### 6因子定义及权重

| 因子名称 | 定义 | 基础权重 | 动态权重范围 | 方向 |
|---------|------|---------|-------------|------|
| **trend** | MA20年化斜率 | 18% | 16-20% | 正向 |
| **momentum** | 20日涨幅 | 13% | 11-15% | 正向 |
| **quality** | ROE质量分 | 18% | 16-20% | 正向 |
| **sentiment** | 前一日涨跌幅 | 13% | 11-15% | 正向 |
| **valuation** | -(PE*0.5 + PB*10) | 18% | 16-20% | 反向（负值好） |
| **liquidity** | 成交额对数 | 10% | 8-12% | 正向 |
| **size** | 小市值因子 | 10% | 6-14% | 正向 |

### 因子标准化方法
- Z-score标准化：对每个因子计算 `(值 - 均值) / 标准差`
- 限制在[-3, 3]范围内防止极端值
- 映射到[0, 100]分：`score = 50 + zscore * 15`

### 动态权重调整（P1优化）

## 选股流程

### Step 1: 硬性过滤
- 价格范围：5元 ≤ 价格 ≤ 150元
- 成交额：日成交 ≥ 5000万
- 排除ST/*ST/退市/北交所
- 趋势过滤：价格 ≥ MA20 * 0.90 且 MA20向上

### Step 2: 因子计算
- 计算6个因子的原始值
- Z-score标准化
- 映射到0-100分

### Step 3: 因子加权评分
```
综合得分 = Σ(因子分 × 动态权重)
```

### Step 4: 选股规则
- 得分阈值：≥55分
- 最多选股：5只
- 冷却期：3天内不重复选同一只股票
- 行业权重上限：单行业 ≤ 30%

### Step 5: 持仓规则
- 买入：T+1开盘价买入
- 卖出：固定持仓2天后卖出（原来是1天）
- 仓位：90%仓位（根据市场强度动态调整）

## 已知问题清单

### P0级问题（致命）
1. **胜率太低**：45.5% < 54%（覆盖成本所需）
2. **盈亏比差**：平均盈利 < 平均亏损
3. **2024-2025熊市失效**：连续两年亏损
4. **回撤过大**：84.6%超出风控范围

### P1级问题（严重）
1. **trend/momentum高度相关**：共线性问题
2. **sentiment因子质量低**：前一日涨跌更接近噪音
3. **市值因子权重过高**：10%权重可能过度暴露小盘风格
4. **2026年过拟合嫌疑**：收益集中在最后3个月

### P2级问题（次要）
1. **缺乏基本面因子**：无ROE同比、营收增长等
2. **缺少反转因子**：A股反转效应未利用
3. **行业中性化不足**：行业权重约束可能不够
4. **动态权重效果存疑**：市场强度判断准确性未知
