# DeepSeek 选股策略评分模板

## 使用说明

此模板用于**标准化多因子选股策略**的专业评估。重点评估**选股能力（Alpha）**，而非交易执行。

**核心原则**:
- ✅ 关注因子有效性、IC值、选股能力
- ✅ 评估因子组合的科学性
- ❌ 不评估止损、仓位管理、回撤控制（属于交易策略层面）

## 🎯 评分维度（选股策略层面）

请DeepSeek从以下维度评估**选股能力**：

### 1. 策略逻辑评分 (0-10分)
- **因子选择合理性**: 因子是否有经济逻辑支撑
- **因子间独立性**: 是否存在高度共线性(>0.7)
- **评分方法科学性**: 标准化、加权方法是否合理
- **选股规则有效性**: 阈值、数量限制是否合适

### 2. 因子有效性评分 (0-10分)
- **IC值分析**: 各因子IC是否稳定、显著
- **因子方向**: 因子方向是否符合逻辑
- **因子单调性**: 高分组是否持续跑赢低分组
- **因子缺失**: 是否缺少关键因子（如市值因子）

### 3. 回测可信度评分 (0-10分)
- **回测周期**: 是否覆盖完整牛熊周期（至少2年）
- **过拟合风险**: 参数是否过度优化
- **数据质量**: 数据是否完整、准确
- **统计显著性**: 样本量是否足够

### 4. 改进价值评分 (0-10分)
- **相比上一版本**: 是否有明显提升
- **优化方向**: 改进路径是否清晰
- **创新点**: 是否有新的有效因子或方法

## 🔍 特别关注点（选股策略层面）

请DeepSeek特别关注以下问题：

### 1. 过拟合风险评估
- 回测周期是否覆盖完整牛熊周期（至少2年）
- 参数（如阈值、权重）是否经过过度优化
- 收益是否集中在特定时间段（如仅2026年Q1）
- 是否使用未来函数

### 2. 因子有效性检查
- **共线性问题**: trend和momentum是否高度相关
- **IC稳定性**: 各因子IC是否在不同时间段保持一致
- **因子方向**: 如value因子方向是否正确（低PE好还是高PE好）
- **缺失因子**: 是否缺少A股重要因子（如市值、行业）

### 3. 策略逻辑合理性
- 因子是否有经济逻辑支撑
- 因子加权是否合理
- 选股阈值是否合适
- 冷却期设置是否必要

### 4. 回测质量评估
- 数据覆盖范围是否完整
- 股票池定义是否清晰
- 是否存在幸存者偏差
- 交易成本是否已考虑（双边0.2%）

### 5. 对比分析
- 与市场基准（如沪深300）相比如何
- 与随机选股相比是否有显著优势
- 不同市场环境（牛/熊/震荡）下表现是否一致

---

## ❌ 不属于本评估范围

以下内容属于**交易策略层面**，本评估不涉及：
- 止损机制设计
- 仓位管理（动态仓位、回撤减仓）
- 市场环境过滤（熊市空仓）
- 资金曲线优化
- 实盘风控系统
