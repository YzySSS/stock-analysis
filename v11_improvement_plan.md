## 目标
基于渐进式改进原则，在控制风险的前提下，逐步优化策略性能。

## 第一阶段：加入市场环境判断（Regime Switching）

### 1.1 改进内容

**市场状态定义**:
```python
# 趋势市条件
is_bull_market = (
    上证指数_close > MA200 且
    MA20 > MA60
)
```

**策略切换逻辑**:
```
if is_bull_market:
    # 趋势市：执行原版V11策略
    score_threshold = 55
    max_positions = 5
    stop_loss = -0.08
else:
    # 震荡市：降低仓位或暂停
    score_threshold = 60  # 提高门槛，只选最优
    max_positions = 2     # 减少持仓
    stop_loss = -0.05     # 收紧止损
    # 或：直接空仓，不交易
```

### 1.2 预期效果
- 2024年牛市：保持原有盈利（+93%）
- 2025年震荡市：减少亏损或空仓避开

### 1.3 回测验证
- 时间段：2024-2025年
- 对比基准：原版V11策略
- 关键指标：2025年亏损幅度是否降低

## 第二阶段：调整因子权重

### 2.1 改进内容

**当前权重**:
- Turnover: 35%
- LowVol: 35%
- Reversal: 30%

**调整后权重**:
- Turnover: 35%（不变）
- LowVol: 20%（降低15%）
- Reversal: 30%（不变）
- **Quality(ROE): 15%**（新增）

### 2.2 新增质量因子

**定义**:
```python
# ROE行业中性化得分
def calculate_quality_score(stock, date):
    roe = stock['roe']  # 最新ROE-TTM
    if pd.isna(roe):
        return 50  # 默认中等分
    
    # 在行业内排名
    industry = stock['industry']
    peers = get_industry_stocks(industry, date)
    
    if len(peers) < 5:
        return 50
    
    # ROE分位数得分
    roe_rank = pd.Series(peers).rank(pct=True)[stock['code']]
    return roe_rank * 100
```

### 2.3 预期效果
- 增加策略韧性，在震荡市中也能选出质量较好的股票
- 不牺牲太多2024年的盈利

### 2.4 回测验证
- 对比第一阶段结果
- 检查2024年收益是否仍然良好
- 检查2025年亏损是否减少

## 风险控制

### 每次改进的验证标准

1. **2024年收益不低于+80%**（保留策略进攻性）
2. **2025年亏损不超过-30%**（相比原-65%，减少一半）
3. **胜率提升至42%+**（接近盈亏平衡点）
4. **最大回撤不超过-40%**

如果某阶段改进效果不佳：
- 回滚到上一阶段
- 分析失败原因
