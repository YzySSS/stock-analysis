# 选股策略优化计划

## 修改记录

### 当前状态 (2026-03-23)
- 5因子模型：技术25% + 情绪20% + 板块30% + 资金15% + 风险10%
- 问题：因子同质化严重、缺乏趋势确认、无大盘择时
- 今日表现：8只推荐股平均收益-2.84%，成功率12.5%

---

## 优化目标

### 因子权重重构
| 因子 | 当前 | 修改后 | 变化 |
|------|------|--------|------|
| 技术因子 | 25% | **20%** | ↓ |
| 情绪因子 | 20% | **合并/简化** | ↓↓ |
| 板块轮动 | 30% | **35%** | ↑ |
| 资金流向 | 15% | **20%** | ↑ |
| 风险因子 | 10% | **15%** | ↑ |
| 大盘择时 | 无 | **新增** | 阈值控制 |

---

## P0 - 核心修改（立即做）

### 1️⃣ 技术因子增强（20%）
从"当日涨"转向"趋势确认"

**新增指标：**
- [ ] 突破20日高点：+5分
- [ ] 突破60日高点：+5分
- [ ] 站上所有均线（MA5/10/20/60）：+4分
- [ ] RSI在强势区（50-80）：+3分
- [ ] 跌破20日均线：-5分（惩罚）

### 2️⃣ 情绪因子简化
**方案：** 取消独立情绪因子，原权重平移到技术/板块

**简化版指标（如保留）：**
- [ ] 换手率5%-15%：+3分
- [ ] 近期首次涨停/大阳线：+5分

### 3️⃣ 大盘择时模块（新增）
**触发条件：**
- [ ] 沪深300在20日均线下方
- [ ] 或MACD死叉状态

**应对措施：**
- [ ] 总分阈值从65分→75分
- [ ] 或限制选股数量（最多1只）

### 4️⃣ 成交量结构过滤
**替换原"成交量>100万"：**
- [ ] 主板成交额>1亿
- [ ] 中小盘成交额>3000万
- [ ] 换手率3%-25%（排除冷门和过度投机）
- [ ] 量比>1.5

---

## P1 - 重要修改（本周做）

### 5️⃣ 板块轮动深化（35%）
**双周期动量：**
- [ ] 5日动量（短期）权重0.4
- [ ] 20日动量（中期）权重0.6

**板块强度公式：**
```
板块强度 = RPS(5日) * 0.3 + RPS(20日) * 0.5 + 成交额占比变化 * 0.2
```

**龙头识别升级：**
- [ ] 板块RPS前10%：+12分
- [ ] 板块RPS前30%：+8分
- [ ] 超越板块平均2倍：+5分

### 6️⃣ 风险因子扩展（15%）
- [ ] 20日最大回撤>12%：-5分
- [ ] 处于历史高位（80%分位）：-5分
- [ ] 近期股东减持公告：-10分或过滤
- [ ] 限售股解禁<30天：-5分

---

## P2 - 增强修改（下周做）

### 7️⃣ 资金流向精细化（20%）
**需要接入数据源：**
- [ ] 主力净流入占比>10%：+6分
- [ ] 大单主动性买入>60%：+4分
- [ ] 北向资金连续3日净流入：+5分
- [ ] 上涨但主力净流出：-5分（诱多信号）

### 8️⃣ 一致性预期因子（5%，可选）
- [ ] 近3日有机构调研：+3分
- [ ] 券商上调评级：+5分
- [ ] 目标价上调>20%：+3分
