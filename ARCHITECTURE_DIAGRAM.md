```
用户持仓 (7只)
    │
    ├──→ 盘前 08:50 ──→ V8选股(全市场5486只) ──→ TOP3推荐
    │                      │
    │                      └──→ 你的持仓分析
    │
    ├──→ 盘中 12:30 ──→ V8选股更新 ──→ 异动监控
    │
    └──→ 盘后 15:35 ──→ 完整分析
                           │
                           ├──→ 持仓详细
                           ├──→ V8选股
                           ├──→ AI决策仪表盘
                           │       └──→ DeepSeek分析

                           │               ├──→ 买卖点
                           │               └──→ 检查清单
                           └──→ 推荐回顾
                                   └──→ 准确率统计
```
