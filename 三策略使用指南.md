## 📋 概述

系统已整合三种互补的交易策略，根据不同市场环境灵活运用：

| 策略 | 适用市场 | 风险偏好 | 核心逻辑 |
|------|----------|----------|----------|
| **突破策略** | 趋势市 | 激进 | 追涨强势股，买在趋势确立 |

## 🚀 策略1：增强版突破策略

### 核心逻辑
突破20日高点 + 放量 + 板块强势

### 买入条件（全部满足）
- ✅ 股价突破前20日最高价
- ✅ 成交量 > 20日均量 × 1.5
- ✅ 所在板块排名前3
- ✅ RS相对强度 > 70

### 卖出规则
- **止损**: 突破点下方 -3%
- **止盈**: 目标价 +15% 或 跌破10日均线

### 适用场景
- 牛市主升浪
- 板块轮动初期
- 个股启动点

### 使用示例
```python
from src.breakout_screener import BreakoutScreener

screener = BreakoutScreener()
signals = screener.screen_breakout_stocks(
    all_stocks=all_stocks,
    top_sectors=top_sectors,
    max_results=5
)

for signal in signals:
    print(f"买入 {signal.name} @ ¥{signal.current_price}")
    print(f"止损 ¥{signal.stop_loss} | 止盈 ¥{signal.target_price}")
```

## 🔲 策略2：ETF网格交易策略

### 核心逻辑
在价格区间内划分网格，低买高卖

### 适用标的
- 银行ETF (159887)
- 电力ETF (159611)
- 沪深300ETF (510300)
- 中证500ETF (510500)

### 参数设置
```python
# 银行ETF示例
网格区间: ¥1.15 - ¥1.45
网格数量: 6格
每格份额: 约5150股 (30900/6)
```

### 交易规则
| 价格 | 操作 | 数量 |
|------|------|------|
| 跌至买入价 | 买入 | 5150股 |
| 涨至卖出价 | 卖出 | 5150股 |
| 跌破下限10% | 止损 | 全部清仓 |
| 突破上限15% | 止盈 | 全部清仓 |

### 使用示例

## 🔄 策略3：双动量策略

### 核心逻辑
Gary Antonacci的Dual Momentum Investing理念  
个股动量 + 板块动量双重验证

### 买入条件（全部满足）
- ✅ 个股20日涨幅排名前10%
- ✅ 所在板块20日涨幅排名前5%
- ✅ RS相对强度 > 60
- ✅ 成交量 > 20日均量 × 1.2

### 卖出条件（满足任一）
- ❌ 个股动量排名掉出前30%
- ❌ 板块动量排名掉出前20%
- ❌ 跌破10日均线

### 优势
- 避免弱势板块中的假强势
- 提高胜率，减少震荡市假信号
- 顺势而为，买在板块主升浪

### 使用示例
```python
from src.dual_momentum_screener import DualMomentumScreener

screener = DualMomentumScreener()
scores = screener.screen(
    all_stocks=all_stocks,
    sectors_data=sectors_data,
    max_results=5
)

for score in scores:
    if score.signal == '买入':
        print(f"买入 {score.name} 得分{score.composite_score}")
        print(f"个股排名: {score.stock_momentum_rank} 板块排名: {score.sector_momentum_rank}")
```

## ⚠️ 风险控制

### 单一策略风险

| 策略 | 最大风险 | 应对措施 |
|------|----------|----------|
| 突破策略 | 假突破 | 严格-3%止损 |
| 网格策略 | 单边行情 | 跌破下限10%止损 |
| 双动量 | 趋势反转 | 排名掉出即卖 |

### 组合使用原则
1. **不重复建仓**: 同一只股不同时在多个策略中持有
2. **分散配置**: 三个策略仓位比例建议 4:3:3
3. **动态调整**: 根据市场环境调整策略权重

## 📈 绩效评估

### 跟踪指标
- **胜率**: 盈利交易次数 / 总交易次数
- **盈亏比**: 平均盈利 / 平均亏损
- **最大回撤**: 资金曲线最大跌幅
- **夏普比率**: 风险调整收益
